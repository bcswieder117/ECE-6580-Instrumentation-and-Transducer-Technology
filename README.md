# ECE 6580: Instrumentation and Transducer Technology


**Course Description**: A study of applications of instrumentation systems, transducer and sensor devices, signal conditioning and recording considerations with emphasis on parameters as temperature, velocity, acceleration, pressure, and others. Calibration techniques, error consideration, and new and current instrument developments will be presented.

This will be an archive of all work done for this course. Taken in Spring 2026 @ Tennessee Tech.
